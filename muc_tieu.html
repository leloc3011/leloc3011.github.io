Mục tiêu ngắn hạn (0-6 tháng)

  • Hoàn thiện 3 dự án cá nhân để đưa lên GitHub.
  • Thành thạo CSS Layout (Flexbox, Grid) và JavaScript cơ bản.
  • Tìm 1 vị trí thực tập part-time hoặc hỗ trợ dự án web.

Mục tiêu trung hạn (6-24 tháng)

  • Học và làm việc thành thạo React (hoặc framework tương đương).
  • Hiểu cơ bản backend (Node.js, Express) và RESTful API.
  • Tham gia dự án nhóm, biết quy trình làm việc với Git và task management.

Mục tiêu dài hạn (2-5 năm)

  • Trở thành lập trình viên Full-stack hoặc Frontend Senior.
  • Đóng góp cho 1-2 dự án mã nguồn mở.
  • Tạo ra sản phẩm/ứng dụng phục vụ người dùng thực tế.

Milestones (Mốc đánh giá)

  1. 3 tháng: 2 project hoàn thiện + CV online
  2. 6 tháng: Có ít nhất 1 thực tập/part-time hoặc freelance nhỏ
  3. 12 tháng: Thành thạo 1 framework frontend, biết deploy cơ bản
