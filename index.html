Giới thiệu nhanh

Xin chào — mình là Lê Xuân Lộc. Mục tiêu của blog này là lưu giữ hành trình phát triển kỹ năng lập trình web, chia sẻ các dự án thực tế và ghi lại những bài học, tips trong quá trình học.

Điểm nổi bật

  • Ngành: Công nghệ thông tin (CNTT 19-01)
  • Chuyên môn đang theo: Frontend (HTML/CSS/JavaScript), bắt đầu học React
  • Dự án mẫu: Portfolio cá nhân, blog tĩnh, trang giới thiệu sản phẩm

Bạn sẽ tìm thấy gì trên blog này?

  1. Bài viết chi tiết về các môn đã học và cách áp dụng vào dự án thực tế.
  2. Hướng dẫn từng bước (tutorial) dành cho người mới bắt đầu web.
  3. Danh sách dự án, source code mẫu và roadmap học tập.

Lời mời

Nếu bạn muốn hợp tác hoặc trao đổi kiến thức, vào trang Liên hệ để kết nối nhé!

Tóm tắt các bài viết gần đây

1. Từ HTML cơ bản đến layout responsive

Tổng hợp các kỹ thuật HTML/CSS để tạo layout thích ứng với mobile và desktop, kèm ví dụ code.

2. Làm project portfolio bằng GitHub Pages

Hướng dẫn deploy trang cá nhân lên GitHub Pages, cấu trúc folder và tối ưu SEO cơ bản.

3. Học lập trình theo lộ trình 6 tháng

Roadmap chi tiết, bài tập thực hành và các nguồn học miễn phí/ trả phí mình đã dùng.
