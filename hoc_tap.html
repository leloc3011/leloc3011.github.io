Danh sách môn học chính

  • Lập trình Web: HTML, CSS, JavaScript — thực hành responsive layout, form, validation.
  • OOP (Java/Python): Biết viết class, kế thừa, đóng gói, xử lý exception đơn giản.
  • Cơ sở dữ liệu: Thiết kế schema, SQL cơ bản (SELECT, JOIN, GROUP BY).
  • Mạng máy tính: Kiến thức TCP/IP, mô hình OSI, bảo mật mạng cơ bản.

Dự án thực tế (chi tiết)

1 — Portfolio cá nhân

Mô tả: Trang tĩnh giới thiệu bản thân, danh mục dự án, liên hệ. Đã deploy lên GitHub Pages.

Công nghệ: HTML, CSS (Flexbox, Grid), GitHub Pages

Bài học: Quản lý repository, cấu trúc folder hợp lý, tối ưu ảnh, metadata cho SEO.

2 — Blog tĩnh

Mô tả: Hệ thống bài viết tĩnh (không dùng CMS) với phân trang đơn giản.

Công nghệ: HTML, Markdown, scripts build đơn giản.

Bài học: Tổ chức nội dung, chuyển đổi markdown sang HTML, tối ưu bài viết để đọc tốt trên mobile.

3 — Trang giới thiệu sản phẩm (mock)

Mô tả: Trang danh mục sản phẩm, giỏ hàng giả lập (front-end), các animation nhỏ.

Công nghệ: JavaScript DOM manipulation, localStorage để lưu giỏ hàng tạm thời.

Roadmap học tập 6 tháng

  1. Tháng 1–2: Hoàn thiện HTML/CSS — Responsive, Grid, Flexbox, basic accessibility.
  2. Tháng 3: Học JavaScript cơ bản — DOM, event, fetch, promises.
  3. Tháng 4: Làm project nhỏ kết hợp API (ví dụ: danh sách phim, thời tiết).
  4. Tháng 5: Học Git nâng cao, deploy, CI/CD cơ bản.
  5. Tháng 6: Bắt đầu tìm hiểu framework (React) và làm 1 project Fullstack đơn giản.

Tài nguyên học tập

  • FreeCodeCamp, MDN Web Docs — tài liệu cơ bản
  • Coursera, Udemy — khóa có cấu trúc
  • GitHub — lưu code, học từ source open-source

Tips học nhanh

  • Code mỗi ngày dù chỉ 30 phút.
  • Làm project thực tế, không chỉ xem tutorial.
  • Chia nhỏ bài tập — focus on problem solving.
  • Tham gia cộng đồng, hỏi khi bí (StackOverflow, reddit, group Facebook học tập).
